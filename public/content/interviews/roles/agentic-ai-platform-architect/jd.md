# Agentic AI Platform Architect

- **Role:** Agentic AI Platform Architect
- **Location:** Pune & Bangalore
- **Experience:** 12–16 years
- **Job Type:** Full Time Employment

## Summary

Design, build, and scale enterprise-grade agentic AI platforms. Drive architecture
for multi-agent systems, orchestrate intelligent workflows, and establish AI-first
engineering practices across the organization. Shape next-generation platforms that
integrate LLMs, enable automation at scale, and embed governance through
Human-in-the-Loop (HITL) systems.

## What You'll Do

- Design and implement end-to-end Agentic AI platform architectures
- Define scalable patterns for multi-agent orchestration, task decomposition, and
  planner-executor models
- Own memory, context, and state management
- Architect secure, enterprise-grade AI platforms with governance, compliance, and
  observability in mind
- Build and optimize multi-agent systems using modern frameworks
- Design advanced orchestration strategies for task delegation, coordination, and
  stateful, resilient execution
- Implement workflow orchestration with tools like Temporal IO (or equivalent)
- Develop core platform capabilities: agent lifecycle management, multi-model
  orchestration and routing, and API integrations with enterprise ecosystems
- Build reusable SDKs and internal platforms for scaling AI adoption
- Design reusable prompt frameworks and agent interaction patterns
- Optimize latency vs accuracy, token utilization, and cost
- Support integration of multiple LLM providers through abstraction layers
- Define evaluation frameworks for agent performance, output quality, and reliability
- Build feedback loops and improvement pipelines
- Implement governance including HITL workflows, guardrails, and audit mechanisms
- Integrate agentic platforms with enterprise systems (CRM, ERP, DevOps tools, data
  platforms)
- Drive adoption of AI-first SDLC practices; enable automation across the development
  lifecycle (coding, testing, documentation)
